Methodology

This challenge was particularly tricky for me due to the simple nature of the task made substantially more difficult by a set of unique constraints, particularly using no javacript!

I solved the task using complex :has() css selector rules, please find some snippets below for easy skimming

Additionally I could not come to a pure html/css only solution to solve the problem of dynamic paragraph line detection so I used a .short class to the paragraph under 10 lines to simulate the desired behaviour
